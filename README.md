# EdgeCAM
This is the official repository for the paper "Exploring Weakly-Supervised Image Manipulation Localization with Tampering Edge-based Class Activation Map". 
This repo provides codes (publicly available upon acceptance of the paper) and our proposed labels for PS-Battles datasets.
As this dataset was created using PS-battles, you must follow the licensing terms of PS-Battles. You can download dataset on https://github.com/dbisUnibas/PS-Battles.
Download our labeled mask data from https://drive.google.com/file/d/10E5Trc3r_ScE4VeyrZ0tGLKM7HTCu3xt/view?usp=sharing
